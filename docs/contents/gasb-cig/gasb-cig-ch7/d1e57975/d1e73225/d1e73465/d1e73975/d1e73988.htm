7.105.1.Q—Statement 34 requires at least one complete condition assessment before the modified approach is permitted. Can the condition assessment be performed in the same year that the modified approach is adopted or the infrastructure acquired prior to the adoption of Statement 34 is reported?
A—Yes. The condition assessment need only be available and documented prior to reporting infrastructure assets using the modified approach.
