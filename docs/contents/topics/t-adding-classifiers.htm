Adding classifiers

Filter data sources by class.
A classifier creates conditions for filtering and classifying data sources. If no conditions are specified, all of the data sources are considered for classification. You can filter data sources by an expression (email, Peer to Peer), protocol or subscriber attribute, for example. These filters are also called "classifier conditions".
  1. Click the Classifiers icon.
    ADDITIONAL INFORMATION: All Classifiers that have been defined will be accessible for selection in the Classifiers view.
    STEP RESULT:
    The Classifiers view opens.
  2. To add a classifier, click Classify.
  3. Enter a name and description for the classifier.
  4. Select the classifier type:
    1. Enumerated: All possible values of the classifier must be specified in the definition
    2. Integer: Classifier can be assigned any integer value
    3. String: Classifier can be assigned any string value.
  5. Click Next.
  6. Click Create a new rule.
  7. Set up one or more rules for the classifer, then click Next.
    ADDITIONAL INFORMATION:
    Rules are conditions that, if met, set values for the classifers.
  8. (Optional) Set target(s) for the classifier (which nodes this classifer is available for).
    ADDITIONAL INFORMATION:
    If no targets are set, then the classifier is available for all elements on all nodes.
  9. Define additional Classifiers by selecting Customize Classifiers option.
    $AdminTask SetMetadataProperty  {--}$AdminTask SetMetadataProperty  {-valueString -f%}
  10. Remove old alarms and filter match items.
    ADDITIONAL INFORMATION: It is a best practice to remove old alarms or filter match items when they are no longer needed. Failure to regularly remove old alarms or filter match items may result in inconsistent reporting results.
    1. Click Filter match items.
    2. Select Filter match items that no longer apply and that you wish to remove.
    3. Click the Delete button.
      STEP RESULT: The old alarms and filter match items will be removed.
  11. Click Finish.
ResultClassifiers have been added to data sources. They can be used to filter and analyze data received from these sources.
Note: You can follow the same procedures to change existing Classifiers or to add new ones.
