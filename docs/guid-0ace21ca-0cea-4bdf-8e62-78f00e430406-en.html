Add  administrators

  • You must be the  owner.
  • The user that you want to add must be a Security adminstrator in . For instructions to add administrators, see Create an administrator in the  Administration content.
  1. On the menu bar, click Org Connect > Administrators.
  2. On the Administrators page, click the Assign role icon.
  3. In the Add an administrator dialog box, search for a user. Click the user that you want to add as an administrator.
    The Assign role to user dialog box opens.
  4. In the Role drop-down list, click the role that you want to assign to the user.
  5. Click Save.
