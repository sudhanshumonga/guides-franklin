Other Material
Generally, all the supporting material in the original Implementation Guides that is still relevant has been updated and included in the respective chapters of this Comprehensive Implementation Guide. Each chapter includes two appendices. Appendix A identifies the source and status of each question and answer number in the chapter (that is, whether a question and answer has been amended or superseded and, if so, the source of that modification). Appendix B presents nonauthoritative illustrations. A topical index covering all chapters is presented at the end of the volume.
