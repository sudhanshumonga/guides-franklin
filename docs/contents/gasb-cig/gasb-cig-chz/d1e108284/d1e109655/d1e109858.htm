Chapter Z, Question Z.42.3

Z.42.3.Q—For capital assets accounted for using the modified approach, should the requirement to disclose the carrying amount of idle impaired capital assets be applied to individual components of the network or subsystem or to the network or subsystem as a whole?
A—Governments generally view infrastructure assets accounted for using the modified approach from a network or subsystem perspective, rather than as individual assets. Accordingly, such individual assets accounted for using the modified approach that are impaired and idle generally would not require disclosure. If the network or subsystem accounted for using the modified approach is impaired and idle, disclosure is required.
