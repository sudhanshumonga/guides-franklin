Linked Direct Memory Access Controller (LDMA)

The Linked Direct Memory Access (LDMA) controller features conkeyref: autogen_keys/device_LDMA_feature_CH_NUM channels capable of performing memory operations independently of software. This reduces both energy consumption and software workload. The LDMA allows operations to be linked together and staged, enabling sophisticated operations to be implemented.
